# Usage:
Each article should have this header:
```
---
layout: article
chapter: "the name of current chapter"
order: "article number in book index"
title: "the name of article"
---
```
Here's an example:
```
---
layout: article
chapter: اصول و مقدمات
order: 1
title: فلسفه آزادی نرم‌افزار
---
```
